A comprehensive Student Information System designed with offline-first, air-gapped architecture for maximum security and uninterrupted workflow. Manage student records, enrollment, grades, attendance, and parent information in one powerful platform.

Complete School Management, Offline-First

Heronix SIS is engineered for educational institutions that demand reliability, security, and operational independence.

Built with air-gapped, offline-first architecture, our comprehensive Student Information System ensures school operations continue uninterrupted regardless of internet connectivity. All critical functions including student records management, enrollment, grades, and attendance work seamlessly without requiring constant online access.

Maximum Data Security

Air-gapped architecture protects sensitive student and staff data from external threats.

Zero Downtime Operations

Work continuously without internet dependency or connectivity interruptions.

Advanced Attendance System

Three flexible modes to fit your school's workflow

Manual Entry Mode

Traditional classroom attendance with teacher/staff entry. Full tardy management system with configurable grace periods and automated parent notifications.

  • Teacher-initiated attendance
  • Tardy tracking and management
  • Configurable attendance codes
  • Audit trail for all entries
Recommended

ID Card Scanning Mode

Fast attendance via student ID card scan. Eliminates manual entry entirely and speeds up the attendance process. Works with existing ID card systems.

  • Instant scan-to-record
  • Eliminates manual entry errors
  • Works with existing ID systems
  • Real-time dashboard updates

Smart Hall Pass Mode

Same ID card system works for hall passes. Student scans ID, credentials auto-populate, selects destination. Fully monitored, logged, and reported to parents daily.

  • Scan ID to auto-populate
  • Select destination from list
  • Real-time location monitoring
  • Daily parent report emails
