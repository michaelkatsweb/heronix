Heronix Hub

Single Sign-On launcher for all Heronix applications. Log in once, access everything. 100% offline authentication with no cloud dependencies or per-user licensing fees.

Explore Features Request Demo

How Hub Works

One Login. All Applications.

1. User Logs In

User opens Hub desktop application and enters username/password. Credentials validated locally with BCrypt security.

2. Token Generated

Hub generates a JWT token valid for 8 hours (full school day) and saves it to a shared file location.

3. Launch Apps

User clicks product tiles (SIS, Scheduler, etc.). Each app reads the token and auto-authenticates - no re-login needed.

Key Benefits

Why Hub Changes Everything

No Cloud Authentication

Unlike Okta or Auth0, Hub doesn't require internet connectivity or third-party authentication services. Everything runs locally on your network.

No Per-User Licensing

Unlimited users with no per-seat SSO fees. Unlike commercial SSO providers that charge per user per month, Hub is included with your Heronix license.

Complete Privacy

Authentication happens entirely on your local network. No credentials or tokens are sent to external authentication providers.

Simple Integration

Other Heronix products automatically detect Hub tokens. No complex SAML or OAuth configuration required.

Technical Specifications

Built for Security and Reliability

Authentication

  • File-based JWT tokens
  • 8-hour session duration (full school day)
  • BCrypt password hashing
  • Automatic token cleanup on logout
  • Role-based access (admin, teacher, student, parent)

Technical Stack

  • Language: Java 21
  • Framework: Spring Boot 3.2.0
  • UI: JavaFX 21
  • 100% offline operation
  • Cross-platform (Windows, Linux)

Included with Heronix Core Suite

Hub is included free with your Heronix SIS or Scheduler purchase. No additional licensing required.
